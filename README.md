# warehouse-map
Updating data:

Open https://joshmc43.github.io/warehouse-map on desktop
Click ↑ Update → load new SAP B1 export
Click ↓ Export → downloads warehouse_data.json
Go to GitHub repo → click warehouse_data.json → three dots → Upload file (or delete and re-upload)
Commit → wait 1 minute → everyone's phone updates automatically

Sharing:
Just send the URL https://joshmc43.github.io/warehouse-map — works on any device, any browser.
